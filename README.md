## Build Setup

``` bash
# clone project
$ git clone https://github.com/kasim393/NFT-Market-MERN.git

# install dependencies server
# > cd api
$ npm install

# install dependencies client
# > cd client
$ npm install

# serve with host at localhost:3000
$ npm start
```
